# High Street Laundromat v2 (Mock Template)

## Run
```bash
cd highstreet-v2
python3 -m http.server 8081
```
Then open `http://localhost:8081` (or tunnel this port from server to Mac).

## Includes
- Conversion-first homepage
- Service + pricing sections
- Reviews and trust block
- Lead capture form
- AI concierge demo widget
